### Required:
    - Nodejs v14
    - Mysql v5.7 or higher
    - Redis v7.2 or higher
### Install:
```npm install -g yarn``` 
```npm install -g nodemon``` 
```yarn``` 
### Config:
- rename ```.env.example``` to ```.env``` and add require config in this
### Run debug or start: 
```yarn dev``` or ```yarn start```
